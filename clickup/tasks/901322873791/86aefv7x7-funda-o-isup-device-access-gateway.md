## Descrição
Objetivo
Criar a fundação técnica ISUP, permitindo que câmeras Hikvision se registrem via Device ID + Key, mantenham sessões persistentes e tenham estado online/offline controlado pela plataforma.

 
Escopo
Serviço ISUP Gateway (TCP)
Porta 7660 (registro)
Autenticação Device ID + Key
Gerenciamento de sessão
Estado online/offline em tempo real
Base técnica para os demais casos de uso


Fora de escopo
Reboot remoto
Eventos de alarme
Configurações
Stream / live view funcional

 

Critérios de aceite
Câmera conecta via ISUP e autentica
Sessão persistente criada
Estado ONLINE/OFFLINE atualizado corretamente
Logs e métricas básicas de conexão
